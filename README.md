# TravelTraces

TravelTraces is a travel-social platform for discovering places, documenting journeys, planning multi-stop trips, and turning completed adventures into story-rich travel records. It is designed around warm editorial storytelling, map-based discovery, community profiles, events, travel groups, and itinerary planning across Southeast Asia.

The product combines a polished landing experience with a logged-in traveller workspace where users can explore destinations, read stories, pin places on maps, plan future routes, join communities, and build printable albums from completed travel plans.

## What TravelTraces Is

TravelTraces helps travellers preserve the full shape of a journey, not just isolated photos or short posts. A user can browse destination guides, read real travel stories, mark locations on a map, create private or public pins, plan routes before travelling, document each stop after visiting, and later publish the completed route as a Travel Plan Story.

The platform is built around one idea: travel memories are stronger when the place, route, people, timing, photos, and story stay connected.

## Core Features

### Landing and Public Website

- Cinematic landing page with TravelTraces branding
- Platform feature sections, featured stories, pricing, help, contact, privacy, terms, and about pages
- Unified warm terracotta, cream, and espresso-brown visual system
- Public-facing pages designed to introduce TravelTraces as a Southeast Asia travel platform

### Explore

- Destination browser for beaches, hikes, food places, hidden gems, forests, culture, and more
- Editorial destination cards with ratings, regions, categories, explorer counts, and coordinates
- Full destination guide pages with long-form travel content
- Explorer stories connected to each destination
- Pin destination flow that sends accurate coordinates to the map

### Stories

- User travel stories focused on personal experiences
- Substack-inspired article reading layout
- Story title, subtitle, category, author, location, posted time, description, photos, and comments
- Story cards and article views with pinned location coordinates
- Save, like, comment, share, and profile navigation flows
- Story pins that can open directly on the map

### Maps

- Private and public map modes
- Pins with categories, notes, photos, coordinates, and visibility rules
- Story, Explore, and Travel Plan routes can send location data into the map
- Travel Plan route display with all route points connected by a visible path
- Smart Meetup Planner and companion-oriented map tools
- Travel marker workflows for creating map-based posts

### Travel Plan Stories

Travel Plan Stories are different from normal Stories. They are multi-destination journeys created from the Draw Route workflow.

- Create a planned route with multiple ordered destinations
- Add a Travel Plan name, cover, description, and destination details
- Set planned day, planned date, planned time, category, notes, and coordinates per point
- Reorder destinations by dragging the itinerary sections
- Prevent travel documentation before the scheduled date and time
- Prevent same-plan time conflicts for destinations on the same date/time
- Document each destination after visiting it with photos and descriptions
- Keep incomplete plans private by default
- Unlock album-related behavior only after all destinations are completed
- Show completed public plans as route-based story content
- View a complete Travel Plan route on the map with every destination point connected

### Album Maker Concept

Travel Plan Stories are designed to become printable travel albums after all destinations are documented. The album system gathers destination photos, titles, descriptions, route order, dates, notes, and journey metadata into a printable travel-book style output.

### Community

- Traveller discovery and profile browsing
- Public visitor-safe profile views
- Follow, message, share, and report-oriented visitor actions
- Gamified user levels, stats, badges, achievements, followers, following, and travel groups
- Community profiles accessible from stories, events, participants, and community lists

### Events

- Travel event listings with Explore/Stories-inspired card styling
- Event detail pages in an editorial event format
- Participants connected to public profile viewing
- Event information, schedules, pricing, location, and participation metadata

### Profile

- Owner profile with profile picture, full name, email, home location, joined date, bio, nationality, level, stats, badges, and settings
- Visitor profile mode that removes owner-only controls and private information
- Calendar section for Travel Plan schedule visibility
- Saved stories and places organized into collections
- Account settings and deletion flows

### Groups and Chat

- Travel groups accessible through chat-oriented navigation
- Group creation flow
- Chat panel with profile-safe behavior
- Group map and travel sharing concepts connected to community travel features

## Current Status

TravelTraces is currently a prototype/product build with a strong frontend experience, local data flows, schema planning, and backend mapping foundations. The next major production step is connecting the frontend workflows to a deployed database and API layer so user actions persist across accounts and devices.
